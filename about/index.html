    About

    This website aims to make it easier to search for scheduled train services. The data that powers this is provided under license by network rail.
    You can access this website on PC/Tablet/Mobile. The website is optimised to work on all devices.
    This website is run in my own spare time and as such I cannot devote 100% of time to it. If there are problems it may take a while to fix, although I would aim to resolve major issues pretty quickly.

    Features

    Home Page

    From the home page you can search for trains:

    • Starting From: a location
    • Calling From: a location, To: a location (Direct Services Only)
    • To: a location (terminate at location)
    • At: a location (call at location en route)
    You can use the "Local" links to lookup stations near to your current location (if permission is granted).
    You can also search for the nearest train to you (useful if on the move). This looks up your current location (if permission is granted) and will look for trains that have passed your location in the recent past.

    Search Results

    From the home page search you are presented with a list of matching trains. You can navigate backwards and forwards in time and then click through to a train in more details.
    Search Results by default hide trains identified as "TOC-ZZ" (Mainly freight), passing trains and Empty Coaching Stock (ECS) trains. You can show these by clicking the "Advanced" option in the top menu bar.

    Train Details

    From the search results page, clicking on a Train Identifier gives you a tabbed layout of train details:

    • Train - lists the planned schedule with current running information included
    • Schedule - lists the planned schedule for the train
    • Live - lists the current running information only
    • Details - lists details about the train, such as operator and type
    • Map - provides a map of planned calling points for this train
    If you click the icon above to activate live tracking then the page will update in real-time with the current data.
    Passing stops are hidden by default. You can show these by clicking the "Advanced" option in the top menu bar.

    Live

    There are a number of live data pages:

    • Routes - This provides live (10 second updates) of train locations on some local routes in the west midlands.
    • Stations - This attempts to provide live (10 second updates) of train locations at Birmingham New Street.

    PPM

    This page provides live (60 second updates) PPM (Public Performance Measure) data of all train operators in the UK. Planned updates include providing historical data lookup

    Related projects

    I also run two other projects based off the data this site generates:

    • traindelay.co.uk - simple interface to see how late your train was
    • twitter.com/fivewaysearly - a twitter account that tweets whenever a London Midland train leaves Five Ways station early

    Accuracy and reliability

    Data Quality

    Some trains can sometimes be missing data, this is often due to the fact the train did not run or did not report properly to the supplied data feed by network rail. Other times it can be do with errors with this website or its associated server side components.

    Server

    This website is hosted via GitHub Pages and the associated server side components that power it are hosted on a Microsoft Azure Server, which should be pretty reliable.

    Technology

    This website and server side componenents are open source and the code for this website can be found on github.
